## Back-end 역활 분배 및 구현 기능
최종민:
  - ERD 설계
  - JWT 토큰 인증 & 인가를 활용한 (로그인 / 회원가입/ 조회) API 구현
  - 화원 상품 리뷰 (작성, 수정, 삭제, 조회) API 구현
  - 에러 핸들 미들웨어 구현

정세한: 
  - ERD 설계
  - 상품 상세 조회 API 구현
  - 상품 장바구니 추가, 수량 조절, 삭제 API 구현

## Back-end 프로젝트 중점사항
